# 環景圖導覽網站

一個互動式的全景圖導覽網站，支援場景切換和家具替換功能。

## 功能特色

- 🏠 **三個場景**：客餐廳、主臥室、次臥室
- 🛋️ **家具替換**：在客餐廳中可以替換沙發和茶几（各3種款式）
- 🎯 **智能點擊檢測**：透過顏色ID圖識別可點選區域
- 🖱️ **滑鼠懸停提示**：可點選區域會顯示指針樣式

## 使用方式

### 場景切換
點擊全景圖中的特定區域（透過ID圖識別）：
- 🩷 **粉色區域** → 切換到客餐廳
- 🟡 **黃色區域** → 切換到主臥室
- 🔵 **藍色區域** → 切換到次臥室

### 家具替換（僅客餐廳）
- 🟢 **綠色區域** → 循環切換沙發（A/B/C）
- 🔴 **紅色區域** → 循環切換茶几（A/B/C）

### 控制面板
右側控制面板提供：
- 場景選擇按鈕
- 沙發和茶几選擇按鈕
- 當前狀態顯示

## 技術說明

- 使用 [Pannellum](https://pannellum.org/) 顯示全景圖
- ID圖用於後台檢測可點選區域（用戶看不到）
- 支援響應式設計，適配各種螢幕尺寸
